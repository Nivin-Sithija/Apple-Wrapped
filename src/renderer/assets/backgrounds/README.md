# Background Images

Place your custom background images in this folder.

## Specifications

- **Resolution**: 390 × 693 px (9:16 aspect ratio)
- **Format**: PNG or JPG
- **File naming**: Use descriptive names, e.g., `crimson-gradient.png`, `gold-gradient.png`

## All-Time Slides (12)

Background images for each all-time slide (optional - gradients used by default):
- `alltime-intro.png` — Intro slide
- `alltime-library-overview.png` — Library overview
- `alltime-top-songs.png` — Top songs list
- `alltime-song-highlight.png` — #1 song highlight
- `alltime-top-artists.png` — Top artists list
- `alltime-artist-highlight.png` — #1 artist highlight
- `alltime-top-albums.png` — Top albums list
- `alltime-genres.png` — Genres
- `alltime-library-journey.png` — Library journey chart
- `alltime-fun-stats.png` — Fun facts
- `alltime-personality.png` — Personality
- `alltime-outro.png` — Outro

## Yearly Slides (10)

Background images for each yearly slide:
- `yearly-intro.png` — Year intro
- `yearly-additions.png` — New additions
- `yearly-top-songs.png` — Top songs list
- `yearly-song-highlight.png` — #1 song highlight
- `yearly-top-artists.png` — Top artists list
- `yearly-artist-highlight.png` — #1 artist highlight
- `yearly-genre.png` — Genre of year
- `yearly-personality.png` — Personality of year
- `yearly-snapshot.png` — Library snapshot
- `yearly-outro.png` — Year outro

## Notes

- If no image is provided, the app uses CSS gradients based on the palette
- Images should be vibrant and high-contrast for Spotify Wrapped aesthetic
